# APF1 - Plataforma de Gestión Inmobiliaria "Bienes Raíces"

# Descripción
Sistema web para la gestión y visualización de propiedades inmobiliarias, 
desarrollado para la Universidad Tecnológica del Perú (UTP) - Curso Integrador II.

# Tecnologías
- **Backend:** PHP 8.1+
- **Base de Datos:** MySQL 8.0+
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Control de Versiones:** Git + GitHub
- **Editor:** Visual Studio Code
- **Diseño:** Figma
